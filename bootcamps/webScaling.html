Web Scaling Workshop

Introductory Workshop on Building Scalable Web Applications

Description

  • Introductory workshop on web architecture for scalable web applications
  • Discuss business requirements for scalable web applications
  • Discuss key technical challenges in scalable web applications
  • Discuss core concepts and solutions to address these challenges
  • Understanding Scaling Solutions provided by AWS, Google Cloud Platform (partly hands-on sessions)
  • Scaling in Practice - how companies like FB, Google, Twitter etc scale their services




    • Who is this workshop useful for?


    • Upcoming software engineers / web programmers looking to get a bigger picture of web architecture, design and scaling problems in the web application world.
    • Current software engineers looking to level-up on scaling / web arch skills.
    • Startup CTOs looking to scale their technology stack in the near future OR building their first technology solution and want to keep future scalability in mind



    Pre-Requisites

    • Some prior web programming experience (in any language / framework of choice)
    • Basic understanding of how web apps (and the Internet) work - Request-Response Cycle, Client-Server Model, HTTP Protocol, Databases.



    Dates and Duration

    • Date : 29th April 2017 Saturday
    • Duration  : 3 Hours
