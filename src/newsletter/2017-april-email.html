ANDROID DEVELOPERS' DAY
 

Coding Blocks, in association with Nagarro, hosted Android Developers’ Day on April 1st in the premises of the Nagarro in Gurgaon. John Souchak (Program Manager, Google Mountain View) and Akash Shukla (Program Manager, Android Developer Skilling, Google India) interacted with the students and Professional Android Developers. Even some CB team members – Arnav, Umair, Harshit, Aayush and others shared their Google Summer of Code (GSoC) experience with the students aspiring for GSoC and helped them in writing the proposals for the same. Over two hundred students and developers attended the event.

LAUNCH OF BOSS
 

Bountiful Open Source Summer (BOSS) is a summer program by Coding Blocks to encourage students to participate in open source software development and strengthen the open source community in India. Uber BOSS, students can contribute to a variety of softwares which are a part of the Coding Blocks Git Hub repository, and win bounties for their contribution. Students accumulating the maximum bounty will win Macbooks, smartphones and other exciting prizes.

WEB SCALING WORKSHOP

 

The Web Scaling Workshop was a one day workshop conducted by Ankit Malik, an IIT Delhi graduate who has earlier worked at Facebook. He covered various topics related to scaling web server applications such as web architecture and business requirements for scalable web applications as well as some core concepts. Various techniques used to scale web backends to handle the demands of growing user base were also covered. The workshop was attended by students as well as working professionals.
CODERS' EVENING @ DTU

 

Mentors from Coding Blocks and some bright DTU students gathered at DTU’s renowned snack point, Maggi Baba. GAME DEV BOOTCAMP

 

Coding Blocks, in association with Nagarro, conducted an online bootcamp on Game Development to help students learn and develop games using popular game development engines like HTML5, Canvas and Phaser. This bootcamp, conducted by Mr. Manmohan Gupta and Mr. Prateek Narang, was held between April 8 and 16 and it introduced the participants to JavaScript and HTML5. The instructors covered games such as Pokemon Shooter, Mario, Flappy Birds and many more. All participants thoroughly enjoyed the bootcamp and developed amazing games.
