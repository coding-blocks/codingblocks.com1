DEV-LOVERS WEEKEND HELD
SOCKET.IO
The Socket.Io workshop was conducted by our in-house app development expert, Rishabh Khanna. It started with an introduction to Node.js, and covered the concepts of client server architecture, real time apps and why we need them. The workshop also covered a hands-on session on building a server using Express.js and making it real time using Socket.IO. It ended with a fully functional chat web app and a real-time music controller web app. 
PYTHON FOR WEB
The Python for Web Codelab was conducted by one of the countrys top web authorities on Python, Nikhil Kumar Singh aka Indian Pythonista. It was all about interacting with web using Python, from the basics all the way till sending mails using Python. Some of the topics that were covered were introduction to web APIs, working with GET and POST requests, downloading large files from web, web scraping, web automation and sending mails.
API DEVELOPMENT USING JAVA
The Java API Development workshop was conducted by GSoCer and GSoC-GCI org-Admin for JBoss, Anuj Garg. The objective of the workshop was to get students started into web app development using Java. The workshop started with an introduction to dependency management and build tools such as Maven. Anuj explained the use of Spring Boot and Rest API development to make a small restful HTTP service and a to-do list using that service.
GAME DEVELOPMENT WEBINAR
The Game Development Webinar was conducted by Prateek Narang, one of the Coding Blocks founding members and was held on the Coding Blocks YouTube channel. Meant for students with basic programming abilities, the webinar covered Javascript and HTML5 Canvas from the basics by building an exciting game. The webinar started with the basics and went on to explain CSS and ended with building a simple game.
CODING BLOCKS MENTORS ATTEND GDD
In December, CB mentors Aayush Arora and Harshit Dwivedi were invited to the Google Developer Day India event by Google in Bangalore. The event had speakers from Google talking about various technologies like Android Web, AI and Machine Learning. The event lasted for two days with over thirty workshops, talks and codelabs featuring speakers like JP Souchak (Program Manager for Certification), Florina Muntenescu (Dev Advocate for Google), Lyla Fujiwara (Android Dev Advocate for Google) and Dan Galpin (Dev Advocate for Google)
JBOSS WORKSHOP @ GREATER NOIDA
Coding Blocks Greater Noida conducted a three-day workshop on Open Source development titled Winter of Code with JBoss Community. It was conducted by CB Mentor and GSoC/Google Code-In organization-admin for JBoss, Anuj Garg. The workshop was meant to get more students on to the open source bandwagon, by explaining to them the concepts and methodology of open source development. Students were also told the steps to approach a new server/client/CLI project, approaching open source organizations and applying for competitions.
